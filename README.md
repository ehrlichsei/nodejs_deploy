## run locally
```
npm start
```
or
```
node index.js
```

or for automatic deployment after each change
```
nodemon index.js
```

## configure
```
npm install
```

## Test
```
curl -X POST http://localhost:3000/submit -H "Content-Type: application/json" -d '{"name": "John Doe", "age": 30}'
```
